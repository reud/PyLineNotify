## Usage
You can connect with LINE more easily!

if you need to get token, Access here !

This code is to send  'Hello World!' to your LINE.
```python
import pylinenotify

pylinenotify.send_message(token='YOUR_LINENOTIFY_TOKEN',message='Hello World!')

```
